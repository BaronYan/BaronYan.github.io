变量名与 PHP 中其它的标签一样遵循相同的规则。

  • 一个合法类名以字母或下划线开头，后面跟着若干字母，数字或下划线。

PHP 中的变量用一个$符号后面跟变量名来表示。

  • PHP 中的变量名是区分大小写的。
  • PHP 中的所有标签命名遵循相同的规则：
    • 一个有效合法的标签名以字母或者 _ 开头，后面跟上任意数量的字母，数字或 _ 。
