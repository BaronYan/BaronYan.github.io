---
title: "link 标签详解"
---
## 使用场景
HTML 中 link 元素规定了外部资源与当前文档的关系。  
这个元素最常于链接样式表。   
还能用来创建站点图标(比如PC端的“favicon”图标和移动设备上用以显示在主屏幕的图标)。
## 用法
要链接一个外部的样式表，你需要像这样在你的 head 中包含一个 link 标签。
```html
<link href="main.css" rel="stylesheet">
```
在这个例子中，使用了 `href`  属性设置外部资源的路径，  
并设置 `rel`属性的值为“stylesheet”。  
`rel` 的主要功能是表明“关系”——被包含文档与当前文档的关系。

- **内容类型**	元数据。
- **允许的元素内容**	无，这是一个空元素。
- **标签省略**	鉴于这是一个空元素，开始标签必须存在，结束标签必须不存在。
- **允许的父元素**	任何可以接受元数据的元素。
- **DOM接口**	HTMLLinkElement
## 属性
这个元素可以使用 全局属性.
- as 仅在`link`设置了 rel="preload" 时有效。它规定了加载的内容的类型。
- crossorigin 枚举，指定在加载相关图片时是否必须使用 CORS.  可取的值如下:
    - anonymous 起一个跨域请求(即包含 Origin: HTTP 头). 但不会发送任何认证信息. 如果服务器没有给出源站凭证 (不设置 Access-Control-Allow-Origin: HTTP 头), 这张图片就会被污染并限制使用.
- disabled 用来启用或禁用多个样式表链接
- href 被链接资源的URL
- hreflang 被链接资源的语言
- integrity 包含行内元数据，它是一个你用浏览器获取的资源文件的哈希值，以base64编码的方式加的密，这样用户能用它来验证一个获取到的资源,在传送时未被非法篡改
- media 外部资源适用的媒体类型
- rel 此属性指明被链接文档对于当前文档的关系
- sizes 这个属性定义了包含相应资源的可视化媒体中的icons的大小。它只有在rel包含icon的link类型值。它可能有如下的规则。
- type 定义链接的内容的类型。
